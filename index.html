Copia de seguridad
¿Por qué hacer una copia?
Tus datos se guardan automáticamente en este navegador cada vez que agregas o modificas un registro. Sin embargo, si limpias el caché del navegador o usas otro dispositivo, los datos podrían perderse. Descarga una copia de seguridad periódicamente para proteger tu información.
Exportar

El archivo .json guarda todos los datos incluyendo comentarios y puede restaurarse aquí.
El archivo .xlsx es solo para ver en Excel, no se puede restaurar.

Restaurar copia

Carga un archivo .json exportado previamente desde esta aplicación. Puedes reemplazar todos los datos actuales o combinarlos con los nuevos.
